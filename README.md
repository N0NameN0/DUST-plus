# 🚀 DUST-Plus

**DUST-Plus** is a productivity-focused Chrome extension for the [Dust.tt](https://dust.tt) platform. It helps power users manage their recurring prompts through a sleek, integrated interface.

## ✨ Features

- **📚 Prompt Library**: Manage an unlimited number of pre-defined prompts.
- **📍 Smart Insertion**: Pastes your selected prompt exactly at your cursor position, preserving your workflow.
- **✏️ Inline Edition**: Edit your prompts directly in the list (no annoying popups).
- **🗑️ Safe Deletion**: Two-click deletion system to avoid accidental removals.
- **🎨 Native Integration**: Perfectly matches Dust's Spark design system, including Dark Mode and smart positioning (Top/Bottom).
- **🛡️ Privacy First**: Everything is stored locally in your browser via `chrome.storage`. No external servers, no tracking.

## 🛠️ Installation

1. Clone this repository (or download the zip).
2. Go to `chrome://extensions/` in your browser.
3. Enable **Developer Mode**.
4. Click **Load Unpacked** and select the DUST-Plus folder.
